# [Week 1] PyTorch入門 + 深層学習基礎

## 学習目標

- PyTorch で何ができるかを理解する
- 深層学習の基本概念（順伝播・逆伝播・最適化）がわかる
- 既存モデル（YOLO等）を動かせるイメージが持てる
- 簡単なネットワークを自分で書いて学習させられる

## トピック一覧

### 1. PyTorch とは何か
- **概念**：NumPy の GPU版 + 自動微分エンジン + ニューラルネットフレームワーク
- **何ができるか**：テンソル計算・ネットワーク定義・学習・既存モデル活用
- **図解アイデア**：PyTorch のレイヤー構造（torch / torch.nn / torch.optim の関係）
- **研究との接点**：D435 の点群データや画像をテンソルとして扱う

### 2. テンソルとは何か
- **概念**：多次元配列。スカラー・ベクトル・行列の上位概念
- **何ができるか**：画像・音声・センサデータを統一した形式で扱える
- **図解アイデア**：0次元〜4次元テンソルの視覚化（画像バッチの例）
- **研究との接点**：D435 の深度マップ = 2D テンソル、RGB画像 = 3D テンソル（H×W×C）

### 3. 自動微分（Autograd）
- **概念**：計算グラフを自動構築し、誤差逆伝播を自動計算する仕組み
- **何ができるか**：手動で微分を計算せず、`loss.backward()` の1行で勾配が求まる
- **図解アイデア**：計算グラフの可視化（y = ax + b の場合）
- **研究との接点**：ノイズモデルのパラメータ推定に応用可能

### 4. ニューラルネットワークの構造（nn.Module）
- **概念**：`nn.Module` を継承してネットワークを定義する
- **何ができるか**：全結合・CNN・RNN・Transformer 全部同じ枠組みで書ける
- **図解アイデア**：nn.Module の継承ツリー、forward() の流れ
- **研究との接点**：LeRobot のモデルも全部 nn.Module で定義されている

### 5. 学習ループの全体像
- **概念**：データ読み込み → 順伝播 → 損失計算 → 逆伝播 → パラメータ更新
- **何ができるか**：任意のタスクに対して学習パイプラインを組める
- **図解アイデア**：学習ループのフローチャート（1エポックの流れ）
- **研究との接点**：RL の学習ループも基本構造は同じ

### 6. 損失関数と最適化
- **概念**：損失関数（MSE・CrossEntropy）でモデルの「悪さ」を定量化、最適化器（SGD・Adam）でパラメータを更新
- **何ができるか**：タスクに合わせた損失関数を選べる
- **図解アイデア**：損失曲面と勾配降下のアニメーション（簡易版）
- **研究との接点**：ノイズモデルのフィッティングも最小二乗法 = MSE 最小化

### 7. 既存モデルの活用（torchvision・YOLO）
- **概念**：学習済みの重みをロードして推論・ファインチューニングに使う
- **何ができるか**：YOLO で物体検出、ResNet で画像分類、などを数行で実現
- **図解アイデア**：転移学習の概念図（特徴抽出層 vs ファインチューニング）
- **研究との接点**：温室内の植物・果実検出に YOLO を活用できる
