# physical-ai-learning

松尾研 Physical AI 基礎編講座（2026年）の学習教材リポジトリ。

PyTorch・深層強化学習・世界モデルの概念を、**日本語の図解付き HTML** で学習するための資料を管理します。

## 構成

| ディレクトリ | 内容 |
|---|---|
| `pytorch/` | PyTorch入門・深層学習基礎 |
| `reinforcement-learning/` | 強化学習基礎（Silver講義・Stanford CS234） |
| `world-model/` | 世界モデル（Dreamer等） |
| `physical-ai/` | 松尾研講座ノート |

## 学習フロー

```
Phase 1（〜6/24）  : PyTorch入門 → 強化学習基礎
Phase 2（6/25〜8/6）: 松尾研講座と並走
Phase 3（8〜9月）  : 最終課題・苦手補強
```
